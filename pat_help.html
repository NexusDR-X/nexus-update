pat Installation/Upgrade Help

Released Version vs. "With Forms" Version

pat is currently available via 2 releases. The released "production" version has no support for forms. A separate version of pat that supports Winlink Forms handling is also available. It uses the same forms as the Winlink Express Windows program.

If you currently have the released version of pat installed and you select "pat with forms", the released version will be replaced with the "with forms" version. Likewise, if you have the "with forms" version installed and you select the released version, the released version will replace the "with forms" version. In either case, your configuration settings and any messages you have stored on this Pi will be retained.

"pat With Forms" Information

The "with forms" version designates folder $HOME/.wl2k/Standard_Forms as the location in which the Winlink forms will be stored. This folder will automatically be created if it doesn't already exist.

Once "pat with forms" is installed, follow these steps to download/update the winlink.org forms to this Pi:
  1. Click Raspberry > Hamradio > Direwolf TNC and pat GUI to start Direwolf and pat.
  2. Use the Configure tabs to configure Direwolf and pat as desired.
    • Make sure you've checked Start pat web service when Direwolf TNC starts in the Configure pat tab.
    • Click Saves Settings and Restart Direwolf + pat if you've made and changes.
  3. Click Open pat Web interface.
  4. In the pat web interface, click Action, then Update Form Templates.
